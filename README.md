# Landing Page de Boda - Temática de Viaje

Este proyecto es una landing page moderna para invitaciones a una boda, con temática de viaje.

## Estructura
- `src/index.html`: Página principal.
- `src/style.css`: Estilos.
- `src/script.js`: Lógica RSVP.
- `src/assets/plane.png`: Imagen de avión (placeholder).

## Personalización
Puedes cambiar los textos, colores y agregar imágenes propias en la carpeta `assets`.

## Visualización
Abre `src/index.html` en tu navegador.
